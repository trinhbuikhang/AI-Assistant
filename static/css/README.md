Edit these files to change the look of the app. Load order matters: variables first, then layout and components, mobile last.

| File | Purpose |
|------|--------|
| **variables.css** | Colors (--accent, --bg-*), typography base, reset. Start here for theme changes. |
| **layout.css** | App flex container, main content area. |
| **sidebar.css** | Left sidebar: logo, New Chat button, conversation list, model selector. |
| **header.css** | Top bar: title, model badge, clear/settings buttons. |
| **messages.css** | Message scroll area, vignette, scrollbar styling. |
| **welcome.css** | Empty state: avatar, greeting, suggestion chips. |
| **message-bubbles.css** | User/assistant bubbles, code blocks, copy button, typing dots. |
| **input.css** | Input area, file chips, folder panel, textarea, send button, error toast. |
| **mobile.css** | Sidebar toggle, responsive rules (@media max-width: 768px). |
